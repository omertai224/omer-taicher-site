# CLAUDE.md - הנחיות עבודה עם קלוד

## תהליך פתיחת סשן עבודה חדש

כשפותחים סשן חדש (אחרי מחיקת הבראנץ' הישן):

### שלב 1 - יצירת סשן עבודה
המשתמש כותב: **"עבודה"**
קלוד יוצר בראנץ' עבודה חדש ודוחף אותו.

### שלב 2 - דחיפת העץ
המשתמש כותב: **"דחוף את העץ"**
קלוד דוחף את הבראנץ' ל-GitHub.

### שלב 3 - עדכון מועדף "מיזוג" (ידני)
המשתמש נכנס ל-GitHub, מעתיק את קישור ה-PR של הבראנץ' החדש, ומעדכן את המועדף בדפדפן שנקרא **"מיזוג"** עם ה-URL החדש.
המיזוג הוא המקום שבו ממזגים בין ה-Preview (GitHub Pages) לבין הפרודקשן.

### שלב 4 - הפעלת GitHub Pages (ידני)
המשתמש נכנס ל-Settings בפרויקט ב-GitHub → Pages → משנה את הבראנץ' של GitHub Pages לבראנץ' החדש שנוצר.

### שלב 5 - צפייה ב-Preview
רק אחרי שלב 4, המועדף **"Preview"** יעבוד ויציג את השינויים מהבראנץ' החדש דרך GitHub Pages.
