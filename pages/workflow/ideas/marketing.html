הרעיון בקצרה

כל שאלה שחוזרת בקבוצת הפייסבוק = פוסט בבלוג + קישור להדרכה. תוכן שלא צריך להמציא — הוא כבר קיים בשיחות עם לקוחות.

למה זה יעבוד
  • שאלות שחוזרות = ביקוש מוכח לתוכן הזה
  • קל לייצר — השאלה כבר קיימת, רק צריך לכתוב תשובה מפורטת
  • כל פוסט = עוד דף שמדורג בגוגל = עוד תנועה
  • Claude Code יכול לכתוב את הפוסטים מתמלולי שיחות
איך ליישם
  • רשימה של 10-20 שאלות שחוזרות (מקבוצת פייסבוק + תמלולי שיחות)
  • כל שבוע: בחירת שאלה → כתיבת פוסט עם Claude Code → פרסום
  • כל פוסט מסתיים ב-CTA להדרכה רלוונטית

הרעיון בקצרה

אין דף שמציע הרצאות לחברות וארגונים. עמוד אחד עם 3-4 נושאים + טופס פנייה = הזמנות ב-2,000-5,000₪ להרצאה.

למה זה יעבוד
  • הרצאות לארגונים = הכנסה גבוהה (2,000-5,000₪ להרצאה)
  • הניסיון כבר קיים — רק צריך עמוד שמציג את זה
  • ארגונים מחפשים בגוגל "הרצאה על אבטחת מידע לעובדים" — SEO קל
  • עמוד אחד פשוט + טופס פנייה = עלות אפסית
