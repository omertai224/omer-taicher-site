קבצי JSON שחשוב להכיר

קבצי JSON הם "מקור האמת" של האתר. הם שולטים על מה שמוצג.

קובץמה הוא שולט
interactive.jsonכל ההדרכות: שמות, מחירים, תיאורים, דרישות, סטטוס
posts-index.jsonרשימת כל הפוסטים בבלוג (בלי התוכן עצמו)
posts/xxx.jsonתוכן של פוסט בודד (כותרת, תקציר, גוף, SEO)
vercel.jsonהגדרות שרת: ראוטינג, headers, cache, functions
CLAUDE.mdהנחיות לקלוד: כללי ברזל, דברים שחשוב לזכור
איך הכל מתחבר
1
GitHub - כל הקוד נמצא שם. Claude Code דוחף שינויים לבראנץ', ואתה ממזג ל-main.
2
Vercel - עוקב אחרי main. ברגע שממזגים, Vercel בונה גרסה חדשה ומעלה לאוויר תוך דקה. גם מריץ את ה-API (תשלומים, webhook, בלוג).
3
GitHub Pages - משמש לתצוגה מקדימה (Preview). מציג את הבראנץ' הנוכחי לפני שממזגים. גם משמש את פאנל הניהול.
4
omertai.net - הדומיין שמצביע על Vercel. זה מה שהלקוחות רואים.
